Теорія циклів Python

Цикл while

Коли використовувати while?

  • Коли НЕ відома кількість повторень
  • Коли є умова виконання
  • Коли цикл залежить від логіки, а не від range()
            i = 0
            while i < 3:
                print(i)
                i += 1

Як працює while?

Цикл while виконується, поки умова є True. 

            while умова:
                код

Якщо умова стає False — цикл зупиняється.

Приклад з умовою

            i = 1
            while i <= 3:
                print(i)
                i += 1

Нескінченний цикл

Якщо не змінювати змінну — цикл може стати нескінченним. 

            i = 0
            while i < 3:
                print(i)
                # ❌ забули i += 1

Такий код буде виконуватись безкінечно!
