📘 Теорія циклів Python

🔹 Цикл for

🔹 Коли використовувати for?

  • Коли відома кількість повторень
  • Коли потрібно пройтись по списку
  • Коли є послідовність значень
 for i in range(3): print(i)

🔹 Як працює range()

Функція range(n) створює послідовність чисел від 0 до n-1. 

        range(5) → 0,1,2,3,4

🔹 ... range(start, stop)

        for i in range(1, 4):
            print(i)

🔹... range(start, stop, step)

Функція range() може мати три параметри:

  • start — з якого числа починаємо
  • stop — до якого (не включно)
  • step — крок (через скільки збільшувати)

🔹 Приклад

        for i in range(0, 10, 2):
             print(i)
